utils.encryption module¶

utils/encryption.py

Moduł odpowiedzialny za hashowanie i weryfikację haseł użytkowników z wykorzystaniem biblioteki bcrypt.

utils.encryption.check_password(password: str, hashed: str) → bool[źródło]¶

Weryfikuje, czy podane hasło odpowiada wcześniej wygenerowanemu hashowi.

Parametry:

  • password – hasło w postaci jawnej (string)

  • hashed – hasz hasła, wygenerowany wcześniej przez hash_password (string)

Zwraca:

True, jeśli hasło jest poprawne, w przeciwnym razie False

utils.encryption.hash_password(password: str) → str[źródło]¶

Generuje bezpieczny hasz z podanego hasła.

Parametry:

password – hasło w postaci jawnej (string)

Zwraca:

hasz hasła zakodowany jako string

utils.statistics module¶

utils/statistics.py

Moduł statystyk nawyków.

Zawiera:

  • calculate_max_streak: funkcję obliczającą najdłuższą serię dni, w których nawyk był wykonany,

  • load_habits: metodę do załadowania nawyków użytkownika do widżetu Combobox.

utils.statistics.calculate_max_streak(logs: Sequence[HabitLog]) → int[źródło]¶

Oblicza najdłuższą serię kolejnych dni, w których nawyk został wykonany.

Parametry:

logs – sekwencja obiektów HabitLog zawierających pola .date oraz .completed_at

Zwraca:

długość najdłuższej serii dni z rzędu, gdy .completed_at nie jest None
